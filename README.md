# R6-ACE

Mod for the game Rainbow Six: Siege

## How to build

you need:

[DirectX Software Development Kit](https://www.microsoft.com/en-us/download/details.aspx?id=6812)

[Visual Studio](https://visualstudio.microsoft.com/pl/downloads/)

Build the project in Release configuration.

## Docs

This mod uses Direct3D9 for render.

Features:

- **See all other players through objects**
- **Automatically aim at target player**
- **Remove movement modifiers from the weapon**
